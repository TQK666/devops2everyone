## Windows

* Download aws cli:
http://docs.aws.amazon.com/cli/latest/userguide/awscli-install-windows.html

* to install awscli
`
python -m pip install awscli
`

* to upgrade
`
python -m pip install --user --upgrade awscli
`

* to configure aws

```
aws configure
```
